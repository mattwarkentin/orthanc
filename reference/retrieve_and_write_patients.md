# Retrieve and write patients to given path

Retrieve and write patients to given path

## Usage

``` r
retrieve_and_write_patients(patients, path)
```

## Arguments

- patients:

  List of
  [Patient](https://mattwarkentin.github.io/orthanc/reference/Patient.md)s

- path:

  Path where you want to write the files.

## Value

Nothing, invisibly.
